EngKey

A real-time desktop translator that runs on your computer. Type in any language, get instant translations using Google, DeepL, Microsoft, OpenAI, or LibreTranslate. No internet browser tabs needed — just a clean, always-available floating window.

Un traductor de escritorio en tiempo real que funciona en tu computadora. Escribe en cualquier idioma y obtén traducciones al instante usando Google, DeepL, Microsoft, OpenAI o LibreTranslate. Sin necesidad de pestañas del navegador, solo una ventana flotante limpia y siempre disponible.
