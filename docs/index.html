Installation

Linux

chmod +x EngKey-Linux-x86_64 && ./EngKey-Linux-x86_64

Or via .deb: sudo dpkg -i engkey_1.2.0_amd64.deb

macOS

chmod +x EngKey-macOS-x86_64 && ./EngKey-macOS-x86_64

Windows

Double-click EngKey-Windows-x86_64.exe to run.

From Source

Requires Python 3.12+

git clone https://github.com/Vashuus/EngKey.git
cd EngKey
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python3 engkey.py
