EngKey

A real-time desktop translator that lives in your computer. Type in any language, get instant translations using Google, DeepL, Microsoft, OpenAI, or LibreTranslate. No browser tabs, no clutter — just a clean, always-available window.

Un traductor de escritorio en tiempo real que vive en tu computadora. Escribe en cualquier idioma y obtén traducciones al instante usando Google, DeepL, Microsoft, OpenAI o LibreTranslate. Sin pestañas del navegador, sin desorden — solo una ventana limpia y siempre disponible.
